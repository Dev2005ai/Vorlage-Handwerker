# Vorlage: Handwerker Demo

## Projektbeschreibung
Demo-Website-Vorlage für Handwerksbetriebe (Elektriker, Maler, Sanitär, Schreiner etc.). Wird als Basis für Kundenprojekte verwendet.

## Tech Stack
- Pure HTML + CSS + JavaScript (kein Framework)
- Mobile-first, DSGVO-konform, lokale Fonts

## Design-Regeln
- Nutze das AskUserQuestion Tool, um den Nutzer über das Websitedesign zu interviewen
- Nutze den frontend-design Skill für alle UI-Entscheidungen
- Nutze UI/UX Pro Max für Design-System-Generierung
- Keine generischen AI-Aesthetics — bold, distinctive Design-Choices
- Performance-optimiert (Core Web Vitals)
- Vertrauen & Seriosität vermitteln, klare Handlungsaufforderungen

## Zielgruppe
Elektriker, Maler, Sanitär/Heizung, Schreiner, Dachdecker, Fliesenleger, Gebäudereinigung

## Typische Sektionen
- Hero mit starkem CTA (Kostenlos anfragen / Jetzt anrufen)
- Leistungen (Übersicht der Gewerke)
- Warum uns wählen (USPs: Festpreise, 24h-Service, Garantie)
- Referenzen / Vorher-Nachher Fotos
- Über den Betrieb (Gründung, Team, Zertifikate)
- Einzugsgebiet / Regionen
- Bewertungen (Google, ProvenExpert)
- Kontakt & Schnellanfrage-Formular

## Typische Designsprache
- Kräftige Farben passend zum Gewerk (Elektro: Gelb/Schwarz, Maler: Weiß/Bunt, Sanitär: Blau/Weiß)
- ODER seriöses Dunkelblau/Grau mit oranger Akzentfarbe
- Klare Sans-Serif Fonts (Vertrauen, Lesbarkeit)
- Große CTAs, Telefonnummer immer sichtbar
- Professionell ohne übertrieben fancy zu wirken

## Kundenanpassungen
Beim Erstellen eines Kundenprojekts aus dieser Vorlage:
1. Gewerk und Farben anpassen
2. Leistungsliste mit echten Angeboten befüllen
3. Bilder durch echte Referenzfotos ersetzen
4. Einzugsgebiet (Städte/Stadtteile) eintragen
5. Telefonnummer prominent platzieren (Click-to-Call)
6. Angebotsformular einrichten
